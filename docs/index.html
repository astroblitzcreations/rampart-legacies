What the game is

Rampart Legacies mixes tetromino-style wall building, cannon placement, map voting, castle defense, and combat rounds into one fortress battle loop.

Platforms

Windows and Android builds are planned from the same Godot project, with GitHub releases used for public update drops.

Controls

Keyboard, touch controls, and Bluetooth controllers are supported. The in-game Help screen covers login, menus, map picks, and combat controls.
